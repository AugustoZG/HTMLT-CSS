Principais formatações

Negrito / Destaque

Podemos utilizar o Negrito utilizando esta forma ela apenas deixa em Negrito,
mas tambem podemos utilizar este outro método de Negrito onde não somente deixará em Negrito como também fornece um sentido de Destaque

Itálico / Ênfase

Agora podemos utilizar o Termo em Itálico,
Mas como no Negrito também temos um termo que foge além da forma de Termo em Ênfase

Texto de marcação

O texto pode ser marcado utilizando o Mark,
podemos utilizar outra cor abrindo style dentro de Mark

Tamanho dos textos

podemos deixar um texto grande ou texto pequeno utilizando estes comando.

Texto deletado / inserido

Podem ser marcado um texto como deletado indicando que ele já foi lido pelo usuário.
Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Texto sobreescrito / subscrito

Para inserir coisas do tipo fórmulas x20+3
Para inserir coisas do tipo químico h2O
